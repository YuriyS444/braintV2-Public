Политика конфиденциальности

BRAIN T₀ · Последнее обновление: июнь 2026

Настоящая Политика конфиденциальности описывает, как приложение BRAIN T₀ собирает, использует и защищает информацию пользователей.

1. Кто мы

BRAIN T₀ — интеллектуальная AI система кристаллизации знаний, построенная на теории информации T₀. Разработчик: Юрий Сунуров, Алматы, Казахстан.

Контакт: yurisunur@gmail.com

2. Какие данные мы собираем

При использовании приложения мы можем собирать следующие данные:

  • Вопросы и запросы — текст вопросов которые вы задаёте системе
  • Данные кошелька — публичный адрес криптовалютного кошелька при оплате через USDC
  • История диалогов — для формирования базы кристаллов знаний
  • Технические данные — тип устройства, версия браузера, для улучшения работы приложения

3. Какие данные мы НЕ собираем

  • Геолокация
  • Контакты телефонной книги
  • Фотографии и медиафайлы (кроме тех что вы сами загружаете для анализа)
  • Данные других приложений
  • Биометрические данные

4. Как мы используем данные

  • Для обработки ваших запросов и формирования ответов AI
  • Для накопления базы кристаллов знаний системы
  • Для обеспечения платёжных операций
  • Для улучшения качества системы
  • Для технической поддержки

5. Передача данных третьим лицам

Ваши запросы передаются AI провайдерам (Anthropic, OpenAI, DeepSeek) для генерации ответов. Передача происходит в зашифрованном виде согласно политикам конфиденциальности этих сервисов.

Мы не продаём и не передаём ваши данные третьим лицам в коммерческих целях.

6. Хранение данных

Данные хранятся на защищённых серверах Railway.app. История диалогов хранится в течение 30 дней, после чего автоматически удаляется. Кристаллы знаний хранятся бессрочно в обезличенном виде.

7. Безопасность

Мы используем шифрование HTTPS для передачи данных. Пароли хранятся в зашифрованном виде. Платёжные данные (приватные ключи кошельков) мы никогда не запрашиваем и не храним.

8. Права пользователя

Вы имеете право:

  • Запросить удаление своих данных
  • Получить копию своих данных
  • Отозвать согласие на обработку данных

Для этого напишите на yurisunur@gmail.com

9. Дети

Приложение не предназначено для детей младше 13 лет. Мы не собираем намеренно данные детей.

10. Изменения политики

Мы можем обновлять данную политику. При существенных изменениях уведомим пользователей через приложение. Актуальная версия всегда доступна по этому адресу.

11. Контакты

По вопросам конфиденциальности обращайтесь: yurisunur@gmail.com
